# street

Minimal street photography portfolio. 35mm · Stuttgart.

## Setup

```
pip install Pillow
```

## Usage

1. Drop original JPGs into `src/`
2. Run `python build.py`
3. Commit & push

## Structure

```
config.py              ← site settings (title, email, image sizes)
build.py               ← image processing + HTML assembly
templates/
  base.html            ← HTML skeleton with {{placeholders}}
  style.css            ← all styles
  script.js            ← all client-side logic
src/                   ← originals (gitignored)
photos/
  thumb/               ← 600px previews (auto-generated)
  full/                ← 2000px expanded (auto-generated)
index.html             ← auto-generated by build.py
CNAME                  ← custom domain
robots.txt             ← block image indexing
```

## Config

Edit `config.py` to change site title, subtitle, email, image sizes.

## Domain

Cloudflare DNS → CNAME: `street` → `street-ph.github.io` (DNS only)
GitHub → Settings → Pages → Custom domain: `street.kremenskii.art`
