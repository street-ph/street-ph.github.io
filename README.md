# street

Minimal street photography portfolio. 35mm · Stuttgart.

## Setup

```
pip install Pillow
```

## Usage

1. Drop original JPGs into `src/`
2. Run `python build.py`
3. Commit & push

```bash
# First time
git init
git remote add origin git@github.com:USERNAME/street.git

# Every time you add photos
python build.py
git add .
git commit -m "new photos"
git push
```

## Structure

```
src/           ← originals (gitignored, stay local)
photos/thumb/  ← 600px previews for grid (auto-generated)
photos/full/   ← 2000px for expanded view (auto-generated)
index.html     ← auto-generated by build.py
build.py       ← resize + EXIF + HTML generator
CNAME          ← street.kremenskii.art
```

## Domain

Cloudflare DNS → CNAME record:
- Name: `street`
- Target: `USERNAME.github.io`
- Proxy: DNS only (grey cloud)

GitHub repo → Settings → Pages → Custom domain: `street.kremenskii.art`
